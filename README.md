📊 Sales & Revenue Analytics Dashboard

📌 Overview

This project focuses on analyzing business sales data to uncover insights related to revenue, profit, customer behavior, and overall performance. The dashboard provides an interactive and data-driven view to support better business decision-making.

🎯 Objectives

Analyze overall sales performance and profitability
Track monthly, quarterly, and regional trends
Understand customer segments and behavior
Evaluate the impact of discounts on profit margin
Provide actionable insights through visualization

📈 Key Metrics
Total Revenue: 14.91M
Total Profit: 5.24M
Total Quantity Sold: 55K
Total Orders: 10K
YoY Growth: 6%

📊 Dashboard Features
📅 Monthly Revenue Trend Analysis
🌍 Region-wise Revenue & Profit Distribution
👥 Customer Segment Analysis (Consumer, Corporate, Home Office)
📦 Product Category Performance
📉 Profit Margin vs Discount Analysis
🚚 Delivery Performance by Category
🔮 Revenue Forecasting by Quarter

🔍 Key Insights

Revenue remained stable throughout the year with slight growth trends
Certain regions contributed equally to revenue, with minor variations in profit
Higher discounts significantly impacted profit margins
Corporate and Consumer segments generated the highest revenue
Technology and Office Supplies showed better efficiency in delivery time

🛠️ Tools & Technologies
Power BI – Data Visualization & Dashboard Creation
Microsoft Excel – Data Cleaning & Preparation
SQL – Data Extraction & Querying

📂 Project Structure
Dataset/ – Raw and cleaned data files
Dashboard/ – Power BI dashboard file (.pbix)
Images/ – Dashboard screenshots
README.md – Project documentation

🚀 How to Use
Download the .pbix file from the repository
Open it using Power BI Desktop
Explore the interactive dashboard using filters and slicers
